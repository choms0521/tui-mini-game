# 미니게임 플레이 가이드 (한국어)

터미널에서 즐기는 파이썬 게임 모음입니다. 모든 게임은 워크스페이스 루트의
가상환경 하나를 함께 쓰며, 같은 설계를 따릅니다(불변 게임 상태, 깜빡임 없는
트루컬러 렌더링, 화면 없이 도는 자체 테스트).

게임을 실행하면 화면 오른쪽 패널에 **게임 설명과 조작법이 한국어로** 표시되므로,
플레이 도중에도 무엇을 어떻게 하는지 바로 확인할 수 있습니다. 더 자세한 설명이
필요하면 `h`(또는 `?`) 키를 눌러 화면 중앙에 도움말을 띄우고, 다시 누르면 닫습니다.
(블랙잭은 `h`가 히트이므로 `?`로 도움말을 엽니다.)

> 권장 터미널 크기는 **80×30** 이상입니다. Tron(약 112칸)과 Pong(약 85칸)처럼 넓은
> 게임은 더 큰 창이 필요하며, 창이 작으면 실행 시 크기 안내가 표시됩니다.

---

## 1. 사전 준비

가상환경은 이 저장소 루트에서 **한 번만** 만들면 됩니다.

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

- 파이썬 3.10 이상이 필요합니다.
- 트루컬러를 지원하는 터미널을 권장합니다(macOS 터미널, iTerm2 등 최신 터미널).

---

## 2. 실행 방법

### 방법 A — 런처(권장)

런처는 모든 게임을 한 메뉴에 보여 주고, 고른 게임을 별도 프로세스로 실행한 뒤
게임을 끝내면 다시 메뉴로 돌아옵니다.

```bash
./play.sh
```

런처 조작:

| 키 | 동작 |
| --- | --- |
| 위/아래 방향키 | 게임 선택 |
| Enter | 선택한 게임 시작 |
| `q` | 런처 종료(게임 중에는 메뉴로 복귀) |

### 방법 B — 게임 하나만 바로 실행

각 게임 폴더의 `run.sh`를 실행하면 그 게임만 바로 켜집니다.

```bash
./tetris/run.sh
./connect_four/run.sh
```

---

## 3. 공통 조작

대부분의 게임이 아래 키를 공통으로 씁니다.

| 키 | 동작 |
| --- | --- |
| `q` | 종료 |
| `r` | 재시작 |
| `p` | 일시정지(지원하는 게임) |

각 게임의 세부 조작은 아래 게임별 안내를 참고하세요.

---

## 4. 게임별 안내

### Tetris (`tetris/`)

- **목표**: 블록을 움직이고 회전시켜 가로 줄을 가득 채워 지웁니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 좌우 방향키 | 좌우 이동 |
| 위 방향키 또는 `x` | 회전 |
| `z` | 반대 방향 회전 |
| 아래 방향키 | 천천히 내리기 |
| Space | 즉시 내리기 |
| `p` / `r` / `q` | 일시정지 / 재시작 / 종료 |

### Breakout (`breakout/`)

- **목표**: 패들로 공을 튕겨 모든 벽돌을 깹니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 좌우 방향키 또는 `a` / `d` | 패들 이동 |
| Space | 공 발사 |
| `p` / `r` / `q` | 일시정지 / 재시작 / 종료 |

### Snake (`snake/`)

- **목표**: 먹이를 먹어 몸을 늘리되, 벽과 자기 몸에 부딪히지 않습니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 진행 방향 전환 |
| `p` / `r` / `q` | 일시정지 / 재시작 / 종료 |

### 2048 (`2048/`)

- **목표**: 타일을 밀어 같은 숫자를 합치며 2048을 만듭니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 타일 밀기 |
| `r` / `q` | 재시작 / 종료 |

### Minesweeper (`minesweeper/`)

- **목표**: 지뢰를 피해 모든 안전한 칸을 엽니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 커서 이동 |
| Space 또는 Enter | 칸 열기 |
| `f` | 깃발 표시/해제 |
| `r` / `q` | 재시작 / 종료 |

### Space Invaders (`space_invaders/`)

- **목표**: 내려오는 외계인 함대를 모두 격추합니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 좌우 방향키 | 좌우 이동 |
| Space | 발사 |
| `p` / `r` / `q` | 일시정지 / 재시작 / 종료 |

### Roguelike (`roguelike/`)

- **목표**: 던전을 탐험하며 몬스터를 처치하고 계단(`>`)으로 더 깊이 내려갑니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 이동(몬스터 방향으로 이동하면 공격) |
| `>` 칸 | 닿으면 다음 층으로 하강 |
| `r` / `q` | 재시작 / 종료 |

### Wordle (`wordle/`)

- **목표**: 숨겨진 5글자 단어를 여섯 번 안에 맞힙니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| `a`~`z` | 글자 입력 |
| Backspace | 글자 지우기 |
| Enter | 추측 제출 |
| `r` | 재시작(판이 끝난 뒤) |
| `q` | 종료 |

### Sokoban (`sokoban/`)

- **목표**: 모든 상자를 목표 지점으로 밀어 넣습니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 이동 / 상자 밀기 |
| `u` | 한 수 되돌리기 |
| `r` / `q` | 재시작 / 종료 |

### Pong (`pong/`)

- **목표**: 간단한 AI와 패들 대결을 벌입니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 위 방향키 또는 `w` | 패들 위로 |
| 아래 방향키 또는 `s` | 패들 아래로 |
| Space | 시작 / 일시정지 |
| `r` / `q` | 재시작 / 종료 |

### Connect Four (`connect_four/`)

- **목표**: 원반을 떨어뜨려 가로·세로·대각선으로 먼저 4개를 잇습니다(AI 상대).
- **조작**

| 키 | 동작 |
| --- | --- |
| 좌우 방향키 | 떨어뜨릴 열 조준 |
| Enter / Space / 아래 방향키 | 원반 놓기 |
| `r` / `q` | 재시작 / 종료 |

### Mastermind (`mastermind/`)

- **목표**: 숨겨진 색 코드를 제한된 시도 안에 추론해 맞힙니다. 검은 페그는 색·위치가 모두 맞은 수, 흰 페그는 색만 맞은 수입니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 숫자 키 | 색 선택 |
| Enter | 추측 제출 |
| `r` / `q` | 재시작 / 종료 |

### Sudoku (`sudoku/`)

- **목표**: 9x9 격자에 1~9를 채우되 각 행·열·3x3 박스에 같은 수가 겹치지 않게 합니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 커서 이동 |
| `1`~`9` | 숫자 입력 |
| `0` / Backspace | 지우기 |
| `r` / `q` | 새 퍼즐 / 종료 |

### Reversi (`reversi/`)

- **목표**: 상대 돌을 양끝으로 둘러싸 뒤집고, 끝났을 때 돌이 많은 쪽이 승리합니다(AI 상대).
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 커서 이동 |
| Enter / Space | 착수(합법 수만) |
| `r` / `q` | 재시작 / 종료 |

### Gomoku (`gomoku/`)

- **목표**: 가로·세로·대각선으로 같은 돌 5개를 먼저 이으면 승리합니다(AI 상대).
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 커서 이동 |
| Enter / Space | 착수 |
| `r` / `q` | 재시작 / 종료 |

### Battleship (`battleship/`)

- **목표**: 가려진 적 함대를 사격으로 찾아 모두 격침합니다(AI 상대).
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 추적판 조준 이동 |
| Enter / Space | 사격 |
| `r` / `q` | 재시작 / 종료 |

### Blackjack (`blackjack/`)

- **목표**: 딜러와 겨루어 21에 가깝게, 넘지 않게 만듭니다. 첫 두 장 21(블랙잭)이 가장 강합니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| `h` | 히트(카드 받기) |
| `s` | 스탠드(멈추기) |
| `?` | 도움말 |
| `r` / `q` | 새 판 / 종료 |

### Tron (`tron/`)

- **목표**: 자취(벽)를 남기며 달려 상대를 먼저 충돌시킵니다. 벽·경계·상대와 부딪히면 패배합니다.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 방향 전환(180° 역주행 불가) |
| `p` | 일시정지 |
| `r` / `q` | 재시작 / 종료 |

### Frogger (`frogger/`)

- **목표**: 차도와 강을 건너 위쪽 목표 칸에 도착합니다. 모든 목표 칸을 채우면 클리어.
- **조작**

| 키 | 동작 |
| --- | --- |
| 방향키 | 개구리 이동 |
| `p` | 일시정지 |
| `r` / `q` | 재시작 / 종료 |

## 5. 테스트

각 게임과 런처는 화면 없이 도는 자체 테스트를 갖고 있습니다.

```bash
.venv/bin/python launcher/selftest.py
.venv/bin/python tetris/selftest.py
.venv/bin/python connect_four/selftest.py
```
