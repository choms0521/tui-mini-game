## 1. 개요

- **장르**: 실시간 액션
- **재사용 진법**: Space Invaders/Pong의 실시간 루프 + 충돌 판정
- **한 줄 소개**: 차선과 강을 건너 반대편 목표 지점에 개구리를 무사히 도달시킨다.
- **재미 포인트**: 장애물 사이를 비집고 통나무를 갈아타는 타이밍 액션.

## 2. 규칙

- 화면은 여러 **차선(lane)**으로 구성된다: 안전지대(시작/중간), 도로(자동차), 강(통나무/거북), 목표 지대.
- 도로에서 자동차에 닿으면 사망. 강에서는 물에 빠지면 사망이며, 통나무 위에 올라타면 통나무와 함께 이동한다.
- 통나무에 실린 채 화면 밖으로 밀려나도 사망.
- 목표 지점에 도달하면 점수 획득 후 개구리가 시작 위치로 복귀. 목숨을 모두 잃으면 게임 종료.
- 차선의 장애물은 차선별 방향·속도로 매 틱 스크롤한다.

## 3. 재사용 자산

- 실시간 틱 루프 + 비차단 입력: Space Invaders/Pong `main.py` 패턴.
- 행(차선) 단위 이동 장애물 갱신: Space Invaders 함대 이동의 행 스크롤 사고.

## 4. 상태 모델 (`@dataclass(frozen=True)`)

```
Lane = (kind: str,            # "safe" | "road" | "river"
        direction: int,       # -1 / +1
        speed: int,           # 틱당 이동 칸
        pattern: tuple[int, ...],  # 장애물 시작 배치(차선 폭 기준)
        span: int)            # 장애물 길이(통나무/차량)
GameState:
  frog:        Pos
  lanes:       tuple[Lane, ...]
  offsets:     tuple[int, ...]    # 차선별 누적 스크롤(틱에서 유도 가능, 결정성 위해 보관)
  lives:       int
  score:       int
  tick:        int
  game_over:   bool
  won:         bool
```

- 장애물의 현재 위치는 `pattern + offset`으로 결정적으로 계산(파생). 게임 로직은 순수 함수로 둔다.

## 5. 핵심 판정

- **충돌(도로)**: 개구리 칸이 차량 칸과 겹치면 사망.
- **탑승(강)**: 개구리 칸이 통나무 칸과 겹치면 그 통나무의 `direction*speed`만큼 함께 이동. 강 차선에서 통나무가 없으면 익사.
- **이탈**: 통나무에 실려 격자 밖으로 나가면 사망.
- **도달**: 목표 행 진입 시 점수 + 개구리 리셋.

## 6. 입력

- `화살표`: 개구리 1칸 이동(위로 전진, 좌/우/아래도 가능)
- `p`: 일시정지 · `r`: 재시작 · `q`: 종료
- 입력과 무관하게 틱마다 장애물 스크롤(실시간).

## 7. 렌더

- 차선 종류별 배경/글리프 구분(도로·강·안전지대), 차량/통나무/거북 글리프, 개구리 글리프, 목표 지대 강조.
- 패널: 목숨, 점수, 조작법, 게임오버/클리어 배너. 깜빡임 없는 전체 프레임 1회 출력.

## 8. 셀프테스트 필수 항목

- 개구리 이동이 격자 경계를 벗어나지 않음.
- 목표 도달 시 점수 증가 + 개구리 시작 위치로 리셋.
- 도로 차량 충돌 시 목숨 감소(또는 리셋).
- 강에서 통나무 위는 함께 이동, 통나무 없는 물칸은 익사.
- 통나무에 실려 화면 밖 이탈 시 사망.
- 목숨 소진 시 `game_over`.
- 장애물 위치가 틱에 대해 결정적.
- 불변성.
