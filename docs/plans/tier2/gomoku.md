## 1. 개요

- **장르**: 보드 + AI
- **재사용 진법**: 그리드 렌더 + 패턴 휴리스틱(Connect Four 4목 판정의 일반화)
- **한 줄 소개**: 격자 위에 번갈아 돌을 놓아 가로/세로/대각선으로 5개를 먼저 잇는 쪽이 승리.
- **재미 포인트**: 단순한 규칙 위의 깊은 수읽기. 공격과 방어가 한 수에 얽히는 긴장감.

## 2. 규칙

- 15x15 격자(표준). 사람 vs AI가 번갈아 빈 교점에 돌을 놓는다.
- 가로·세로·대각선 어느 방향이든 같은 색 5개 이상 연속이면 승리(자유형 룰; 렌주 금수는 적용하지 않는다).
- 보드가 가득 차면 무승부(드묾).

## 3. 재사용 자산

- 4방향 라인 스캔은 Connect Four `board.py`의 `DIRECTIONS`/연속 카운트 패턴을 5목으로 확장.
- 격자 렌더/커서 강조: Minesweeper/Connect Four `render.py` 패턴.

## 4. 상태 모델 (`@dataclass(frozen=True)`)

```
GameState:
  board:          tuple[tuple[int, ...], ...]   # 15x15, 0=빈칸, 1=흑, 2=백
  current_player: int                            # BLACK / WHITE
  game_over:      bool
  winner:         int | None                     # 0=무승부
  last_move:      Pos | None
  cursor:         Pos
```

## 5. AI 전략

- **위협 패턴 점수 휴리스틱**: 각 후보 자리에서 자기/상대의 라인 패턴(열린 4, 4, 열린 3 등)을 점수화하여 공격·방어를 동시에 평가.
- 비용 제어: 기존 돌 인접 범위의 후보만 평가하고, 상위 k개 후보에 한해 얕은 미니맥스(깊이 2)를 적용. 동률 수는 주입 RNG로 tie-break.
- 최소 기능 단계에서는 "내 즉시 5목 완성 > 상대 열린 4 차단 > 패턴 점수 최대" 우선순위로 시작.

## 6. 입력

- `화살표`: 커서 이동
- `Enter`/`Space`: 커서 자리에 착수
- `r`: 재시작 · `q`: 종료

## 7. 렌더

- 15x15 격자(교점 표현), 흑/백 트루컬러 돌, 마지막 착수 강조, 커서 강조.
- 큰 보드이므로 셀 폭을 좁게(돌 1~2칸) 잡아 패널과 함께 단일 화면에 맞춘다.
- 패널: 현재 차례, 마지막 수, 승/무 배너.

## 8. 셀프테스트 필수 항목

- 4방향(가로·세로·대각선 2종) 5목 승리 판정.
- 4목에서는 승리로 오판하지 않음.
- 점유된 자리 착수 거부(상태 불변).
- 보드 가득 시 무승부.
- AI는 빈 교점(합법 수)을 반환.
- AI 기초 전술: 자기 즉시 승리 수를 잡고, 상대의 열린 4(즉시 위협)를 차단.
- 불변성, 주입 RNG로 AI 결정적.
